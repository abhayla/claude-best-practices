# Flow File Structure

### Flow File Structure

Organize flows by feature in `e2e/maestro/`:

```
e2e/maestro/
├── auth/
│   ├── login.yaml
│   ├── signup.yaml
│   └── forgot-password.yaml
├── home/
│   ├── home-feed.yaml
│   └── search.yaml
├── settings/
│   ├── profile-edit.yaml
│   └── preferences.yaml
├── navigation/
│   └── tab-navigation.yaml
└── config.yaml
```
